Ellusive Premium Website Template

  • Template Description
  • Template Features

The Ellusive template is a so-called "Multipurpose" template, it hasn't been specifically designed for any type of website. The template can be used for a wide range of different site types, images are a key factor here.

The use of site or product specific images and a bespoke colour scheme help to create an individual template with ease. There is no real restriction with this template, the modular build allows extensive modifications.

As the template comes with both light and dark skins, the possible layout & colour combinations allow for a wide range of personal styling.

The template comes with a modified set of our standard features using its own colours and template specific styling.

The template has been declared as IE9 friendly - but, we use CSS "transitions". The elements that use the transitions are not corrupted when viewed in IE9, they still work, they simply don't have the smooth transition effect.

As standard, the downloadable archive contains two "template" folders: "Full Template" & "Basic Template".

The full template has everything incorporated into it, whereas the basic template simply has the very basics to use the template as a base template, without having to remove all of the predefined HTML code that demonstrates what can be accomplished with the template.

The template comes with a choice of:

  • Responsive - Full Width
  • Responsive - Boxed
  • Light & Dark skin
  • 5 Predefined colour schemes

Page layout types include:

  • Standard - 1, 2, or 3 columns
  • Gallery
  • Blog Overview
  • Blog Single

In use with the framework all of the above layouts can be styled using: Full Width, Sidebar Left, Sidebar Right, Sidebar Left & Right

Extras Included in the downloadable zip:

  • Basic PHP + jQuery Contact Form

Predefined HTML Pages included:

  • Homepage
  • About Us
  • Team
  • Services
  • FAQ
  • Archives
  • 404 Error Page
  • Basic HTML Sitemap
  • Contact
